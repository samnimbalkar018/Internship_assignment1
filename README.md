# Assignment 1: Static Portfolio Website

## Objective
To build a static personal portfolio website using HTML, CSS, and Bootstrap.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## Sections
- Navbar
- Hero Section
- About Me
- Worked With
- Projects
- Contact
- Footer

## How to Run
1. Extract the zip file
2. Open index.html in any browser

## Note
This is a static website. No backend or database is used.
